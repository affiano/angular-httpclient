# angular-starter-httpclient
- Date : 07/12/2018 12:00
- Author : chendra

# Prerequisites
- Angular version 7.1.2
- Angular-cli version 7.1.2
- Visual studio code version 1.29.1
- Typescript version 3.1.6
- node.js version 10.14.1 LTS (Long Term Support)
- npm (node package manager) version 6.4.1

# Installation
- npm install (installing dependencies)
- npm outdated (verifying dependencies)

# Developpement
- npm run start
- http://localhost:4200

# AOT Compilation 
- npm run build
- http-server -p 8080 -c-1 dist/browser
- http://localhost:8080/

# Tests
- npm run lint
- npm run test
- npm run e2e

# SSR Compilation 
- npm run build:ssr
- npm run serve:ssr
- http://localhost:4001/
