# angular-starter-httpclient
- Date : 23/12/2018 09:23
- Author : danny ganatan

# Prerequisites
- Angular version 7.1.4
- Angular-cli version 7.1.4
- Angular universal version 7.0.2
- Visual studio code version 1.29.1
- Typescript version 3.1.6
- node.js version 10.14.1 LTS (Long Term Support)
- npm (node package manager) version 6.4.1
- Jquery version 3.3.1
- Bootstrap version 4.2.1
- Fontawesome version 4.7.0

# Installation
- npm install (installing dependencies)
- npm outdated (verifying dependencies)

# Developpement
- npm run start
- http://localhost:4200

# AOT Compilation 
- npm run build
- http-server -p 8080 -c-1 dist/browser
- http://localhost:8080/

# Tests
- npm run lint
- npm run test
- npm run e2e

# SSR Compilation 
- npm run build:ssr
- npm run serve:ssr
- http://localhost:4000/
